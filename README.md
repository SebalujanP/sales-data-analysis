# 📊 Análisis de Ventas de Tienda Online

Análisis exploratorio de datos sobre un dataset real de ventas con más de 10.000 registros. El objetivo es identificar patrones de rentabilidad y generar recomendaciones accionables para el negocio.

## 🔍 Análisis realizados
- KPIs generales: ventas totales, ganancia, ticket promedio
- Ventas y ganancias por categoría y subcategoría
- Análisis temporal: estacionalidad por mes
- Análisis geográfico por región
- Top 10 clientes por volumen y rentabilidad
- Impacto de descuentos en la ganancia

## 💡 Principal hallazgo
Los descuentos superiores al 30% generan pérdidas. El 18.6% de las transacciones (1.901 ventas) tuvieron ganancia negativa.

## 🛠️ Stack

## 📁 Dataset
Sample Superstore — dataset público de ventas retail.
